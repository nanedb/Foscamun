AUDIO FILES REQUIRED FOR TIMER
================================

This folder should contain the following WAV audio files:

1. tick.wav - A tick-tock clock sound that plays during the last 10 seconds
   - Should be a short sound (0.5-1 second)
   - Plays every second during the countdown
   
2. bell.wav - A bell sound that plays when the timer reaches zero
   - Should be a short bell/alarm sound (1-3 seconds)
   - Plays once when timer expires

You can find free sound effects at:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

Make sure the files are in WAV format for best compatibility.

NOTES:
- The application will work without these files, but no sound will play
- Files must be named exactly as shown above (case-sensitive on some systems)
- Recommended sample rate: 44100 Hz or 48000 Hz
